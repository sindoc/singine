# Multilingual Emotion Scenarios

This pack captures a small multilingual semantic bundle for transliterated
Farsi/Persian care messages aligned across:

- Persian in Latin transliteration (`fa-Latn`)
- Persian script (`fa`)
- English (`en`)
- French (`fr`)

The seed scenario is built around a respectful wellbeing message and models:

- a JSON-LD style knowledge graph context
- outer and inner context dimensions
- temporal causality protection and preservation
- a geometric layer with `S1` cycle semantics and gauge-aware labeling
- data platform references such as Hive, SQL standardization, Docker, and edge servers

The executable fixture is generated from `singine.multilingual_emotion.scenario_fixture()`
and validated by `py/tests/test_multilingual_emotion_bundle.py`.
